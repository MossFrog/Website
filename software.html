Below is a list of software that can come in handy especially during everyday computing.

Most of the software below is free but some may require a lisence or pirated copy.

Sublime Text

Sublime text is a highly versatile and customizable text editor for the Windows operating system. It has been proven to be the best performing and best optimized in contrast to its bloated or outdated opponents such as Atom or Notepad++. Whilst it does prompt for a license in reality it can be used without one just like Winrar, although cracked versions can be found online or it can be purchased for a hefty price.

Link to the website.

Speccy

Speccy is a quick tool for identifying the underlying hardware of any Personal Computer let it be a laptop, server or desktop. It works under the Windows operating system and demonstrates useful information such as hardware manufacturers, operating temperatures and detailed network information.

Download Link.

CCleaner

CCleaner is a simple tool that cleans up unnecessary data off your PC. It is also part of the Piriform software group and has useful features such as disabling applications that run when the system boots and cleaning up the Windows registry. Piriform offers the free version which should suffice for everyday use.

Download Link.

7-Zip

7-zip is a simple compression-decompression tool similar to Winrar yet free as it is licensed under the GNU LGPL license. It has a high compression ratio and supports almost all formats available.

Link to the website.

Microsoft Visual Studio

Visual Studio is a very strong Microsoft based IDE. It offers full C++ Support along with various strong features such as powerful debugging, Windows forms support and networking interfaces. It is also the core development environment for mobile Windows applications and software. Although it is very powerful it comes with a hefty price tag and is generally recommended for professional or academic usage. A good alternative could be Codeblocks or the GNU C Compiler. An earlier free version of Visual Studio is provided by Microsoft and can be used for hobbyist or small scale development, the express version is also provided but is absolute garbage in my personal opinion.

Tiled Map Editor

Tiled is a TMX map creator, highly versatile supporting both layering and collision objects. It is essential if creating a multi level game where lots of fast paced map creation is needed. Due to the widespread usage of the XML format it can be easily parsed by many programming languages such as C++, C#, Python and vice a versa. Along with this the source code is readily accessible on github and proves a worthy opponent for many paid alternatives.

Github link

Official Website

EASy68K

As the name implies this piece of software emulates a Motorola 68000 based system. For those whom want to learn assembly 68K is one of its most basic forms, thus this emulator comes in very handy for grasping basic concepts such as register and memory utilization, bitwise operations and computer microarchitecture. Even though the architecture is outdated it shares many parallels with today's complex systems, not to mention it supports a graphics library allowing users to re-create software for systems such as the Commodore Amiga, Atari ST, TRS-80 and the Sinclair QL!.

Official Website

Deluge

Deluge is a simple "no frills, no ads" peer-to-peer torrent client. Deluge supports a plethora of features such as magnet links, timer based downloads, priority queues and much more making downloads simpler to manage without a licensing fee or useless overhead. Deluge is programmed in python making it compatible with any platform let it be Unix based or a Windows based system. Along with all this its completely free!

Official Download Link

Winamp

Winamp although very old is a versatile audio/video player with strong equalizer technology and a solid library management system. Along with a plethora of skins, visual mods and upgrades Winamp boasts one of the most largest communities on the web alongside Foobar and Mediamonkey. Winamp supports MP3, AAC, M4A, MPC, OGG, FLAC, APE, Opus, TAK, WV, WMA and WAV out of the box making it a quick solution to missing or unreadable file formats.
